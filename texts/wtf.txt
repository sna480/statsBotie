поехали:

1. конторы (гугл, яндекс и прочие) не передают информацию о своих пользователях в интернет через API
API (программный интерфейс приложения) -- такая хуёвина, позволяющая приложениям и сервисам *со всего ✨интернета✨* общаться друг с другом через HTTP/HTTPS протоколы
HTTP -- протокол для передачи инфы в _✨интернете✨_
HTTPS -- его защищённая версия, щас предпочтение отдаётся ему

так как конторы не предоставляют этих данных, онлайн в реальном времени отслеживать *невозможно.*
а сторонних сервисов, которые могли бы такое отслеживать и при этом иметь API, ещё не изобрели (либ за эти данные нужно платить)

==НЕ У ВСЕГО ПОДРЯД ЕСТЬ API!!!==
мой бот может обратиться к API OpenWeather и отправить текущую температуру в Тагиле, потому что API у OpenWeather есть и *оно бесплатное*
но он не может загуглить "скок людей пользуется гуглом" и вернуть текст из первой ссылки, так как *у гугл поиска нет API*


2. хром и сафари являются приложениями, которые лишь предоставляют доступ к _✨интернету✨_
узнать, на каком количестве компов установлено то или иное приложение, тем более *у скольких человек оно открыто прямо сейчас* -- невозможно
